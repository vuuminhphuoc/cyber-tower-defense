## Detailed Specs

### LOG_ANALYZER (Support)

```
Cost:        75
HP:          200
Cooldown:    N/A (passive)
Range:       Global
Effect:      All cloaked threats (APT) within 3 cells become visible
             Cloak duration reduced by 50%
Upgrade 1:   +1 cell reveal radius (cost: 45)
Upgrade 2:   Revealed threats take +10% damage (cost: 75)
```

### SANDBOX (Trap)

```
Cost:        125
HP:          150
Cooldown:    12s
Range:       Single cell (must place on threat path)
Effect:      Threat passing through gets trapped 3s
             Cannot move or attack while trapped
             After release: -20% speed for 5s
Upgrade 1:   Trap duration 3s → 5s (cost: 75)
Upgrade 2:   Trapped threat takes +30% damage (cost: 100)
```

### RATE_LIMITER (Slow)

```
Cost:        100
HP:          250
Cooldown:    N/A (aura)
Range:       2 cells ahead in lane
Effect:      Threats in range move 30% slower
             Does not stack with other Rate Limiters
Upgrade 1:   Slow 30% → 40% (cost: 60)
Upgrade 2:   Also reduces threat attack speed 20% (cost: 90)
```

### ZERO_TRUST_GATE (Wall + Support)

```
Cost:        175
HP:          500
Cooldown:    N/A (wall)
Range:       1 cell aura
Effect:      Blocks threats
             Nearby threats deal 15% less damage to towers
Upgrade 1:   +200 HP (cost: 105)
Upgrade 2:   Aura范围 1→2 cells (cost: 140)
```

### SIEM_CENTER (Global Support)

```
Cost:        225
HP:          300
Cooldown:    N/A (aura)
Range:       Same row
Effect:      Towers in same row deal +25% damage
             Does not stack with other SIEM Centers
Upgrade 1:   +25% → +35% (cost: 135)
Upgrade 2:   Also +15% range to same-row towers (cost: 180)
```

### CLOUD_BACKUP (Recovery)

```
Cost:        150
HP:          200
Cooldown:    180s (after revive)
Range:       2 cells
Effect:      When a tower within range is destroyed,
             auto-revive it with 50% HP
             Only triggers once per Cloud Backup
Upgrade 1:   Revive HP 50% → 75% (cost: 90)
Upgrade 2:   Reduce cooldown 180s → 120s (cost: 120)
```

### QUANTUM_FIREWALL (Shooter)

```
Cost:        350
HP:          350
Cooldown:    1.8s
Damage:      45
Range:       4 cells
Effect:      Projectile pierces up to 3 threats
             Each pierce reduces damage by 20%
Upgrade 1:   Pierce 3→5 (cost: 210)
Upgrade 2:   No damage reduction on pierce (cost: 280)
```

### HONEYPOT_CLUSTER (Chomper + AoE)

```
Cost:        300
HP:          400
Cooldown:    20s (after eat)
Range:       1.5 cells
Effect:      Eats threat under 40% HP instantly
             After eating: AoE slow 40% for 5s in 2-cell radius
             Long cooldown after eating
Upgrade 1:   Eat threshold 40% → 50% (cost: 180)
Upgrade 2:   Reduce cooldown 20s → 14s (cost: 240)
```
