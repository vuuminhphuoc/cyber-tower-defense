## New Bosses

### Botnet Commander (Stage 8 Mini-Boss)

```
HP:        3000
Speed:     0.08
Damage:    30
Special:   - Spawns 3 Botnets on wave start
           - Summons 1 Botnet every 10s
           - Shield active while any Botnet alive
           - Shield reduces damage 60%
Visual:    Large bot icon with satellite dish
Music:     Stage 8 boss theme
```

**Strategy:** Kill Botnets first to drop shield, then DPS boss.

### Satellite Hijacker (Stage 9 Boss)

```
HP:        8000
Speed:     0.06
Damage:    35
Special:   - Disables tower in random lane for 5s
           - "Signal Storm": all projectiles slow 50% for 3s (30s cooldown)
           - Spawns Quantum Worms every 15s
           - Phase 2 (50% HP): double disable frequency
Visual:    Large satellite dish icon
Music:     Stage 9 boss theme
```

**Strategy:** Spread towers across lanes, use Patch Bot to recover disabled towers.

### Quantum Root (Stage 10 Final Boss)

```
HP:        15000
Speed:     0.05
Damage:    50
Special:   - Teleports between lanes every 8s
           - Spawns APT + Rootkit pairs every 12s
           - Has 3 Shield Nodes (must destroy all to damage main body)
           - Phase 2 (30% HP enrage): faster spawns, +50% damage
Visual:    Massive quantum core icon
Music:     Final boss theme
```

**Shield Nodes:**
- 3 nodes at fixed positions
- Each node: 2000 HP
- While any node alive: boss immune to damage
- Nodes regenerate 5 HP/s

**Strategy:** Destroy nodes quickly, manage APT/Rootkit spawns, DPS boss during vulnerability window.

## Implementation Notes

- Add to `config.js` THREAT_TYPES
- Add spawn chances in `waves.js` weighted random
- Shield mechanic: new `_shieldActive` property on threats
- Supply chain aura: proximity check in `entities.js` threat update
- DNS Spoof redirect: modify tower `_target` temporarily
- Quantum teleport: check `cellType` validity before teleport
- Boss phases: track `_phase` and trigger at HP thresholds
