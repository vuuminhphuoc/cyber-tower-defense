# v1.4 — Challenge Modes

## Goal

Increase replay value after campaign completion.

---

## Modes

| Mode | Description | Unlock |
|---|---|---|
| **Endless Mode** | Infinite waves, difficulty scales | Complete Stage 5 |
| **Boss Rush** | Fight bosses back-to-back | Complete Stage 7 |
| **Daily Breach** | Fixed challenge, changes daily | Complete Stage 3 |
| **Minimal Core** | Only 4 towers available | Complete Stage 4 |
| **No Economy** | No Bitcoin Miner, fixed credits | Complete Stage 6 |
| **Speedrun** | Timer + leaderboard (local) | Complete Stage 8 |
| **Fog Nightmare** | Heavy fog + APT focus | Complete Stage 9 |

---

## Endless Mode

### Rules

- Infinite waves with increasing difficulty
- Each wave: +10% threat HP, +5% spawn rate
- Credits earned per kill increase slowly
- Every 10 waves: mini-boss
- Every 25 waves: full boss
- Score = waves survived + threats killed

### Scaling Formula

```js
function endlessScaling(wave) {
  return {
    hpMultiplier: 1 + (wave * 0.1),
    speedMultiplier: 1 + (wave * 0.02),
    spawnRateMultiplier: Math.max(0.3, 1 - (wave * 0.02)),
    creditMultiplier: 1 + (wave * 0.05)
  };
}
```

### UI

- Wave counter: `Wave: ∞ 147`
- Score display: `Score: 12,450`
- High score: `Best: 12,450`
- Timer: `Time: 45:23`

### End Condition

- All lawn mowers used + threat reaches base
- Or player surrenders
- Show final score + stats

## Boss Rush

### Rules

- Fight all bosses sequentially
- 5 bosses total (one per stage)
- Full HP restored between bosses
- Credits carry over between bosses
- Timer tracks total time

### Boss Order

1. Zero-Day Exploit (Stage 7)
2. Botnet Commander (Stage 8)
3. Satellite Hijacker (Stage 9)
4. Quantum Root (Stage 10)
5. Hidden Boss: AI Overlord (unlockable)

## Daily Breach

### Rules

- One fixed challenge per day
- Same challenge for all players
- Local leaderboard (best time / best score)
- Shareable result code

### Challenge Types (Rotating)

| Day | Challenge |
|---|---|
| Monday | Minimal towers, many threats |
| Tuesday | Boss only, no prep time |
| Wednesday | Fog + speedrun |
| Thursday | No walls, all shooters |
| Friday | Random towers each wave |
| Saturday | Double economy, double threats |
| Sunday | Endless until death |

### Daily Seed

```js
function getDailySeed() {
  const today = new Date();
  return today.getFullYear() * 10000 + (today.getMonth() + 1) * 100 + today.getDate();
}
```

### UI

```
[Play Now]
```

## Minimal Core

### Rules

- Only 4 towers available
- Randomly selected at start
- Must complete full campaign (20 levels)
- Score = total credits saved at end

### Tower Selection

```js
function selectMinimalTowers(seed) {
  const all = Object.keys(TOWER_TYPES);
  const rng = seededRandom(seed);
  const selected = [];
  while (selected.length < 4) {
    const idx = Math.floor(rng() * all.length);
    if (!selected.includes(all[idx])) {
      selected.push(all[idx]);
    }
  }
  return selected;
}
```

## No Economy

### Rules

- No Bitcoin Miner available
- Fixed credits at start of each level
- No token collection
- Credits earned only from killing threats
- Score = credits remaining at end

---

## Speedrun Mode

### Rules

- Timer starts on first action
- Timer stops when last threat killed
- Leaderboard tracks best times per level
- Shareable replay code

### Timer Display

```
Speedrun: 12:45.32
Best: 11:23.18
```

---

## Fog Nightmare

### Rules

- 70% fog coverage (vs normal 30%)
- APT spawns 2x more frequently
- Log Analyzer and Scanner cheaper
- Score = waves survived
