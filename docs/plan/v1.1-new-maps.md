# v1.1 — New Maps & Levels

## Goal

Expand campaign from 20 to 30+ levels with new terrain mechanics.

---

## Stage 8: Server Farm

**Theme:** Data center / server racks / cooling systems.

### New Terrain Mechanics

| Terrain | Effect |
|---|---|
| **Server Rack** | Tower placed here gets -15% cooldown |
| **Overheated Zone** | Tower loses 2 HP/s unless Patch Bot nearby |

### Levels

| Level | Waves | Features |
|---|---|---|
| 8-1 | 4 | Introduce Server Rack |
| 8-2 | 4 | Introduce Overheated Zone |
| 8-3 | 5 | Mixed waves + Botnet |
| 8-4 | 5 | APT + Rootkit waves |
| 8-5 | 6 | Mini-boss: Botnet Commander |

### Botnet Commander (Mini-Boss)

- 3000 HP
- Spawns 3 Botnets on spawn
- Summons 1 Botnet every 10s
- Shield active while Botnets alive

## Stage 9: Satellite Network

**Theme:** Space / satellite uplinks / signal disruption.

### New Terrain Mechanics

| Terrain | Effect |
|---|---|
| **Signal Delay Zone** | Projectile speed -30% in this lane |
| **Uplink Node** | Tower here gains +20% range |

### Levels

| Level | Waves | Features |
|---|---|---|
| 9-1 | 4 | Introduce Signal Delay |
| 9-2 | 4 | Introduce Uplink Node |
| 9-3 | 5 | Fog + Satellite interference |
| 9-4 | 5 | Fast threats + DNS Spoofer |
| 9-5 | 6 | Boss: Satellite Hijacker |

### Satellite Hijacker (Boss)

- 8000 HP
- Disables tower in random lane for 5s
- "Signal Storm" slows all projectiles 50% for 3s
- Spawns Quantum Worms

## Stage 10: Quantum Core

**Theme:** Quantum computing lab / final area.

### New Terrain Mechanics

| Terrain | Effect |
|---|---|
| **Quantum Cell** | Changes type (grass/water) every wave |
| **Entangled Pair** | Two cells linked — damage one damages the other |

### Levels

| Level | Waves | Features |
|---|---|---|
| 10-1 | 5 | Introduce Quantum Cell |
| 10-2 | 5 | Introduce Entangled Pair |
| 10-3 | 6 | All threat types |
| 10-4 | 6 | Double boss wave |
| 10-5 | 8 | Final Boss: Quantum Root |

### Quantum Root (Final Boss)

- 15000 HP
- Teleports between lanes every 8s
- Spawns APT + Rootkit pairs
- Has 3 Shield Nodes (must destroy to damage main body)
- Enrage at 30% HP: faster spawns, more damage
