# v3.0 — Modding & Custom Campaigns

## Goal

Allow users to create custom towers, threats, levels, and campaigns via JSON files.
Share mods with community.

## Mod Structure

```
mods/
  my-campaign/
    campaign.json      — campaign metadata
    towers.json        — custom towers
    threats.json       — custom threats
    levels/
      1.json           — level 1
      2.json           — level 2
    README.md          — mod description
```

## Terrain Types

| Type | Effect |
|---|---|
| `grass` | Normal, no effect |
| `water` | Requires Proxy Node |
| `grave` | Threat spawn point, cannot build |
| `server_rack` | Tower cooldown -15% |
| `overheated` | Tower loses 2 HP/s |
| `signal_delay` | Projectile speed -30% |
| `uplink` | Tower range +20% |
| `quantum` | Changes type each wave |
| `lava` | Tower takes 5 HP/s |
| `ice` | Threat speed -20% in this cell |

## Mod Loader

```js
const ModLoader = {
  async loadMod(modPath) {
    const campaign = await fetchJSON(`${modPath}/campaign.json`);
    const mods = {
      campaign,
      towers: {},
      threats: {},
      levels: []
    };

    // Load custom towers
    if (campaign.towersFile) {
      const towers = await fetchJSON(`${modPath}/${campaign.towersFile}`);
      for (const tower of towers) {
        mods.towers[tower.key] = tower;
      }
    }

    // Load custom threats
    if (campaign.threatsFile) {
      const threats = await fetchJSON(`${modPath}/${campaign.threatsFile}`);
      for (const threat of threats) {
        mods.threats[threat.key] = threat;
      }
    }

    // Load levels
    for (const stage of campaign.stages) {
      for (const levelFile of stage.levels) {
        const level = await fetchJSON(`${modPath}/${levelFile}`);
        mods.levels.push(level);
      }
    }

    return mods;
  },

  applyMod(mods) {
    // Merge custom towers into TOWER_TYPES
    Object.assign(TOWER_TYPES, mods.towers);

    // Merge custom threats into THREAT_TYPES
    Object.assign(THREAT_TYPES, mods.threats);

    // Replace level database
    LEVEL_DATABASE.length = 0;
    LEVEL_DATABASE.push(...mods.levels);
  }
};
```

### Import from File

```js
async function importModFromFile() {
  const input = document.createElement('input');
  input.type = 'file';
  input.accept = '.json';
  input.onchange = async (e) => {
    const file = e.target.files[0];
    const text = await file.text();
    const mod = JSON.parse(text);
    saveMod(mod);
    alert(`Mod "${mod.name}" imported!`);
  };
  input.click();
}
```

### Import from URL

```js
async function importModFromURL(url) {
  try {
    const mod = await fetchJSON(url);
    saveMod(mod);
    alert(`Mod "${mod.name}" imported!`);
  } catch (e) {
    alert(`Failed to import mod: ${e.message}`);
  }
}
```

## Mod Validation

```js
function validateMod(mod) {
  const errors = [];

  if (!mod.id) errors.push('Missing mod id');
  if (!mod.name) errors.push('Missing mod name');
  if (!mod.stages || mod.stages.length === 0) errors.push('No stages defined');

  // Validate towers
  if (mod.towers) {
    for (const tower of mod.towers) {
      if (!tower.key) errors.push('Tower missing key');
      if (!tower.cost || tower.cost < 0) errors.push(`Tower ${tower.key} invalid cost`);
      if (!tower.damage && tower.type === 'shooter') errors.push(`Tower ${tower.key} missing damage`);
    }
  }

  // Validate threats
  if (mod.threats) {
    for (const threat of mod.threats) {
      if (!threat.key) errors.push('Threat missing key');
      if (!threat.hp || threat.hp <= 0) errors.push(`Threat ${threat.key} invalid HP`);
      if (!threat.speed || threat.speed <= 0) errors.push(`Threat ${threat.key} invalid speed`);
    }
  }

  // Validate levels
  for (const level of mod.levels || []) {
    if (!level.id) errors.push('Level missing id');
    if (!level.waves || level.waves.length === 0) errors.push(`Level ${level.id} has no waves`);
  }

  return errors;
}
```

## Implementation Notes

- Mod loader fetches JSON via `fetch()` (works with file:// for local files)
- Custom towers/threats merged into existing type objects
- Mod manager UI accessible from main menu
- Mods stored in `localStorage` as JSON
- Future: community mod repository with ratings
- Validation prevents broken mods from crashing game
- All custom content uses same rendering system (emoji/Canvas)
